# 📱 手機相機權限設定指南

## 為什麼需要相機權限？

發票稅務快算使用相機功能來掃描發票，需要您授予瀏覽器訪問相機的權限。

---

## 📋 快速檢查清單

在開始設定前，請確認：

- [ ] 使用 **HTTPS** 連線（相機 API 要求必須使用安全連線）
- [ ] 瀏覽器支援相機 API（Chrome、Safari、Firefox 等現代瀏覽器）
- [ ] 手機系統已授予瀏覽器相機權限

---

## 🔐 各手機系統設定方式

### iPhone (iOS/iPadOS)

#### 方式一：系統設定

1. 開啟 **設定** (Settings)
2. 向下滾動找到 **Safari** 或您使用的瀏覽器（Chrome、Edge 等）
3. 點擊進入瀏覽器設定
4. 找到 **相機** (Camera)
5. 確保開關是 **開啟** 狀態 ✅

詳細路徑：
```
設定 → Safari → 相機 → 開啟
設定 → Chrome → 相機 → 開啟
```

#### 方式二：隱私權設定

1. 開啟 **設定** → **隱私權與安全性** (Privacy & Security)
2. 點擊 **相機** (Camera)
3. 找到您使用的瀏覽器（Safari、Chrome 等）
4. 確保開關是 **開啟** 狀態 ✅

#### iOS Safari 特別注意：

- 首次訪問時，Safari 會自動彈出權限請求
- 如果點選了「不允許」，需要從上述設定重新開啟
- iOS 14.3 以上版本完整支援相機 API

---

### Android 手機

#### 方式一：系統設定

1. 開啟 **設定** (Settings)
2. 找到 **應用程式** 或 **應用程式管理** (Apps)
3. 找到您使用的瀏覽器（Chrome、Firefox、Edge 等）
4. 點擊 **權限** (Permissions)
5. 找到 **相機** (Camera)
6. 選擇 **允許** (Allow) ✅

詳細路徑：
```
設定 → 應用程式 → Chrome → 權限 → 相機 → 允許
設定 → 應用程式 → Firefox → 權限 → 相機 → 允許
```

#### 方式二：瀏覽器內設定

**Chrome：**
1. 在網站頁面上，點擊網址列左邊的 **鎖頭圖示** 🔒
2. 點擊 **權限** 或 **網站設定**
3. 找到 **相機**
4. 選擇 **允許** ✅

**Firefox：**
1. 點擊網址列右邊的 **⋮** (選單)
2. 點擊 **網站資訊**
3. 點擊 **權限**
4. 找到 **相機**，選擇 **允許**

## 🌐 瀏覽器設定

### Google Chrome (桌面版)

1. 點擊網址列左邊的 **鎖頭圖示** 🔒 或 **ⓘ**
2. 點擊 **網站設定**
3. 找到 **相機**
4. 選擇 **允許** ✅

### Safari (macOS)

1. 點擊頂部選單 **Safari** → **此網站的設定...**
2. 找到 **相機**
3. 選擇 **允許** ✅

### Firefox

1. 點擊網址列左邊的 **🔒** 或 **ⓘ**
2. 點擊箭頭 **>** 展開詳細資訊
3. 找到 **權限** 區塊
4. 相機：選擇 **允許**

## 🚨 常見問題排除

### ❌ 「無法存取相機」錯誤

**可能原因：**

1. **權限未授予**
   - 檢查系統設定和瀏覽器設定
   - 重新整理頁面，再次授予權限

2. **相機被其他應用程式占用**
   - 關閉其他正在使用相機的應用程式
   - 重新啟動瀏覽器

3. **非 HTTPS 連線**
   - 確認網址是 `https://`（不是 `http://`）
   - GitHub Pages 自動提供 HTTPS

4. **瀏覽器不支援**
   - 更新瀏覽器到最新版本
   - 使用 Chrome、Safari、Firefox 等主流瀏覽器

### 🔄 如何重新請求權限？

**Chrome/Edge：**
1. 點擊網址列的 **鎖頭** 🔒
2. 點擊 **重設權限**
3. 重新整理頁面

**Safari：**
1. 前往 Safari → 偏好設定 → 網站 → 相機
2. 找到該網站，選擇 **詢問** 或 **允許**
3. 重新整理頁面

**Firefox：**
1. 點擊網址列的 **鎖頭** → 清除權限並重新載入
2. 或前往 設定 → 隱私權與安全性 → 權限 → 相機 → 設定
3. 移除該網站，重新整理頁面

## 💡 使用提示

### ✅ 正常使用流程

1. 訪問網站：https://tw092669-ctrl.github.io/-Invoice-helper/
2. 點擊相機圖示
3. 瀏覽器彈出權限請求 → 點擊 **允許** ✅
4. 相機畫面出現
5. 將發票對準掃描框
6. 點擊拍攝按鈕 📸

### 🎯 最佳實踐

- **光線充足**：在明亮的環境下掃描效果最佳
- **對準框線**：將發票完整放入黃色框線內
- **穩定拍攝**：保持手機穩定，避免模糊
- **清晰可見**：確保發票內容清晰、完整

### 🔄 替代方案

如果相機權限無法開啟，您可以：

1. **從相簿上傳**：
   - 先用手機相機拍攝發票
   - 點擊「相簿」圖示上傳照片

2. **手動輸入**：
   - 直接在主畫面輸入金額

## 🔒 隱私與安全

### 我們如何處理相機資料？

- ✅ **完全在本地處理**：所有影像處理都在您的設備上完成
- ✅ **不儲存影像**：拍攝後立即處理，不會保存照片
- ✅ **僅在使用時啟用**：關閉掃描頁面後立即停止相機
- ✅ **開源透明**：代碼完全開源，可在 GitHub 查看

### 權限說明

相機權限僅用於：
- 📸 掃描發票文字辨識
- 🎯 即時預覽畫面
- 📊 擷取影像傳送給 AI 分析

我們**不會**：
- ❌ 儲存您的照片或影片
- ❌ 上傳到伺服器保存
- ❌ 分享給第三方
- ❌ 用於其他用途

## 📱 支援的平台

### ✅ 完全支援

- iOS 14.3+ (Safari, Chrome, Edge)
- Android 9+ (Chrome, Firefox, Edge)
- macOS Safari 14+
- Windows/Linux Chrome 90+

### ⚠️ 部分支援

- iOS 11-14.2 (功能受限)
- 舊版 Android (需更新瀏覽器)

### ❌ 不支援

- iOS 11 以下
- 非 HTTPS 連線
- 過時的瀏覽器版本
